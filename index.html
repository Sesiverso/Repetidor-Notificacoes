Configurar Notificações Agendadas

Use segundos (ex: 60 para 1 minuto, 3600 para 1 hora)

Esta é apenas a PARTE DO CLIENTE. Ela SALVA sua configuração no servidor. O SERVIDOR precisa estar funcionando para receber os dados e enviar as notificações push agendadas.

Estas notificações funcionam mesmo com a aba fechada, pois são enviadas pelo servidor via serviço Push do navegador. Você precisará conceder permissão para Notificações e Push.
